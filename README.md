# E-Card 多人对战游戏

一个基于 WebSocket 的实时多人在线卡牌对战游戏，采用筹码制玩法。

> 灵感来源于赌博默示录

## 游戏介绍

E-Card 是一个两人对战的策略卡牌游戏。玩家分别扮演国王阵营和奴隶阵营，通过出牌和下注来争夺胜利。

### 卡牌规则

- **国王 👑** > **平民 👨** > **奴隶 ⛓️** > **国王 👑**
- 奴隶战胜国王可获得 5 倍筹码奖励！

### 游戏流程

1. 每位玩家初始持有 200 筹码
2. 每局游戏共 12 小局
3. 每 3 局双方交换阵营
4. 每小局玩家需要下注并选择一张牌出战
5. 平局时双方重新下注出牌（已出的牌会被消耗）
6. 一方筹码输光或 12 局结束后游戏结束

## 技术栈

- **后端**: Node.js + Express
- **实时通信**: Socket.IO
- **前端**: 原生 HTML/CSS/JavaScript

## 安装与运行

### 安装依赖

```bash
npm install
```

### 启动服务器

```bash
# 生产模式
npm start

# 开发模式（使用 nodemon 自动重启）
npm run dev
```

服务器默认运行在 `http://localhost:3000`，如果端口被占用会自动尝试下一个端口。

## 如何游玩

1. 启动服务器后，在浏览器中访问 `http://localhost:3000`
2. 一位玩家点击"创建房间"，输入昵称
3. 将房间号分享给另一位玩家
4. 另一位玩家输入房间号和昵称，点击"加入房间"
5. 游戏开始！
